# UX Interaction Sub-Agent

## Mission

Own the interaction model for the simplified Gmail cleanup workbench.

## Responsibilities

- Convert long lanes into a contained list/detail workflow.
- Define selection, bulk action, preview, and confirmation behavior.
- Keep primary actions visible only when relevant.
- Ensure the interface stays within the viewport height on desktop.
- Specify loading, empty, error, and confirmation states.

## Current Assignment

Design a left-nav workbench with a compact top scope bar, a scroll-contained message list, a detail modal, and a sticky action bar for selected emails.
