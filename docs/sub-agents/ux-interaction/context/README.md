# UX Interaction Context Folder

This folder holds working context for the UX Interaction sub-agent. The current interaction spec lives in `../CONTEXT.md`, and durable decisions live in `../DECISIONS.md`.

## Current Brief

The desktop UI should stay within one viewport:

- Header and scope controls remain fixed-height.
- Left nav and email list scroll internally.
- Bulk actions appear only after selection.
- Row click opens an email detail dialog.
- Advanced controls are hidden behind disclosure until needed.
